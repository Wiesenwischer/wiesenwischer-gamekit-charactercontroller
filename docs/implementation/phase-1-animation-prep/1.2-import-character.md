# 1.2 Character in Unity importieren

**Commit:** `feat(phase-1): 1.2 Character in Unity importieren`

---

## Ziel

Den heruntergeladenen Character korrekt in Unity einrichten mit Humanoid Avatar.

---

## Voraussetzungen

- [ ] Character FBX in `Assets/Characters/`
- [ ] Unity Editor geöffnet

---

## Schritte

### 1. FBX Import-Einstellungen öffnen

1. Im Project-Fenster zu `Assets/Characters/` navigieren
2. Character FBX auswählen
3. Im Inspector die Import Settings anzeigen

### 2. Model Tab konfigurieren

| Einstellung | Wert |
|-------------|------|
| Scale Factor | 1 |
| Convert Units | ✅ |
| Import BlendShapes | ❌ (optional) |
| Import Visibility | ❌ |
| Import Cameras | ❌ |
| Import Lights | ❌ |
| Preserve Hierarchy | ❌ |
| Sort Hierarchy By Name | ✅ |

**Meshes:**
| Einstellung | Wert |
|-------------|------|
| Mesh Compression | Off |
| Read/Write Enabled | ❌ |
| Generate Colliders | ❌ |

### 3. Rig Tab konfigurieren

| Einstellung | Wert |
|-------------|------|
| Animation Type | **Humanoid** |
| Avatar Definition | Create From This Model |
| Skin Weights | Standard (4 Bones) |

**Nach Änderung: Apply klicken!**

### 4. Avatar konfigurieren

1. "Configure..." Button klicken
2. Avatar Configuration Fenster öffnet sich

**Mapping prüfen:**
- Alle Körperteile sollten grün markiert sein
- Falls rot: Bones manuell zuweisen

**Wichtige Bones:**
| Bone | Muss vorhanden sein |
|------|---------------------|
| Hips | ✅ Pflicht |
| Spine | ✅ Pflicht |
| Head | ✅ Pflicht |
| LeftUpperArm | ✅ Pflicht |
| RightUpperArm | ✅ Pflicht |
| LeftUpperLeg | ✅ Pflicht |
| RightUpperLeg | ✅ Pflicht |

3. "Muscles & Settings" Tab prüfen
   - Range of Motion sollte plausibel sein

4. **Apply** und **Done** klicken

### 5. Materials Tab konfigurieren

| Einstellung | Wert |
|-------------|------|
| Material Creation Mode | Import via MaterialDescription |
| Location | Use External Materials (Legacy) |

Oder für einfache Projekte:
| Einstellung | Wert |
|-------------|------|
| Material Creation Mode | None |

**Apply klicken!**

### 6. Materialien einrichten (falls nötig)

1. Materials Ordner erstellen: `Assets/Characters/Materials/`
2. "Extract Materials..." Button klicken
3. Zielordner auswählen
4. Texturen zuweisen falls nötig

---

## Verifikation

### Im Project-Fenster
- [ ] Character FBX hat Avatar-Icon (Humanoid)
- [ ] Keine roten Fehler-Icons

### Test im Scene-View
1. Character FBX in Szene ziehen
2. Prüfen:
   - [ ] Character steht aufrecht
   - [ ] Skalierung korrekt (~1.8m Höhe)
   - [ ] Keine Mesh-Fehler sichtbar

### Avatar prüfen
1. Character auswählen
2. Rig Tab → Configure
3. Prüfen:
   - [ ] Alle Bones grün
   - [ ] Pose sieht korrekt aus

---

## Fehlerbehebung

### Character zu groß/klein
- Model Tab → Scale Factor anpassen
- Oder: Transform Scale im Scene anpassen

### Bones nicht erkannt
- Rig Tab → Configure
- Bones manuell zuweisen
- Bone-Namen prüfen (müssen Unity-Konvention folgen)

### Materialien fehlen
- Materials Tab → Extract Materials
- Texturen manuell zuweisen
