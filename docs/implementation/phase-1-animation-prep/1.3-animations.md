|-----------|-----|--------------|-----------|

| Animation | Typ | Mixamo-Suche | Priorität |
|-----------|-----|--------------|-----------|

| Fall_Loop | Loop | "falling idle" | Pflicht |
| Land | Einmalig | "landing" | Pflicht |

## Mixamo Download-Prozess

### Für jede Animation:

1. **Animation suchen**
   - https://www.mixamo.com → Animations Tab
   - Suchbegriff eingeben (siehe Tabelle oben)

2. **Animation auswählen**
   - Auf Animation klicken
   - Preview im 3D-Viewer prüfen

3. **Parameter anpassen**
   - **Overdrive:** 0 (Standard)
   - **Character Arm-Space:** Anpassen falls Arme durch Körper gehen

4. **Download**
   - Format: **FBX for Unity (.fbx)**
   - Skin: **Without Skin** (wichtig! wir haben bereits den Character)
   - Frames per Second: **30**
   - Keyframe Reduction: **none**

5. **Speichern**
   - In `Assets/Animations/Locomotion/` speichern
   - Benennung: `Anim_[Name].fbx`

## Import-Einstellungen für Animationen

### 1. Animation FBX auswählen

Im Project-Fenster die Animation FBX auswählen.

### 2. Rig Tab

| Einstellung | Wert |
|-------------|------|
| Animation Type | **Humanoid** |
| Avatar Definition | **Copy From Other Avatar** |
| Source | YBot (der importierte Character) |

**Apply!**

### 3. Animation Tab

| Einstellung | Wert |
|-------------|------|
| Import Animation | ✅ |
| Bake Animations | ❌ |
| Resample Curves | ✅ |
| Anim. Compression | Optimal |

**Für Loop-Animationen (Idle, Walk, Run, Sprint, Fall_Loop):**
| Einstellung | Wert |
|-------------|------|
| Loop Time | ✅ |
| Loop Pose | ✅ |
| Cycle Offset | 0 |

**Root Motion (für alle):**
| Einstellung | Wert |
|-------------|------|
| Root Transform Rotation | Bake Into Pose ✅, Based Upon: Original |
| Root Transform Position (Y) | Bake Into Pose ✅, Based Upon: Original |
| Root Transform Position (XZ) | Bake Into Pose ✅, Based Upon: Original |

> **Wichtig:** Root Motion auf "Bake Into Pose" setzen, da wir die Bewegung über den CharacterMotor steuern, nicht über Root Motion.

**Apply!**

## Animation Clips extrahieren (optional)

Für bessere Organisation:

1. Animation FBX auswählen
2. Animation Tab → Clips
3. "+" Button um neuen Clip zu erstellen
4. Start/End Frames anpassen
5. Clip benennen

Oder: Animation Clip direkt aus FBX verwenden.

## Verifikation

### Jede Animation prüfen:

1. Animation FBX auswählen
2. Preview-Fenster unten im Inspector
3. Play Button klicken
4. Prüfen:
   - [ ] Animation spielt korrekt
   - [ ] Keine Verzerrungen
   - [ ] Loop-Animationen loopen sauber
   - [ ] Keine Füße durchdringen Boden

### Animation auf Character testen:

1. Character in Szene
2. Animator Component hinzufügen (temporär)
3. Animation Clip zuweisen
4. Play Mode starten
5. Animation prüfen

---

## Fehlerbehebung

### Animation verzerrt
- Rig Tab → Avatar von Character kopieren
- Prüfen ob Character und Animation gleichen Rig-Typ haben

### Füße schweben/sinken
- Animation Tab → Root Transform Position (Y) → Bake Into Pose

### Animation driftet
- Animation Tab → Root Transform Position (XZ) → Bake Into Pose

### Loop nicht sauber
- Animation Tab → Loop Pose aktivieren
- Start/End Frame anpassen
