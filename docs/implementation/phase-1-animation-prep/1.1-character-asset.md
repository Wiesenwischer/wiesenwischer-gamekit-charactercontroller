# 1.1 Character Asset beschaffen

**Commit:** `feat(phase-1): 1.1 Character Asset beschaffen`

---

## Ziel

Einen geeigneten Humanoid Character für das Projekt beschaffen.

---

## Anforderungen

| Kriterium | Anforderung |
|-----------|-------------|
| Rig | Humanoid (Unity-kompatibel) |
| Pose | T-Pose oder A-Pose |
| Format | FBX |
| Polygone | 10.000 - 30.000 (MMO-geeignet) |
| Texturen | Diffuse, Normal (optional) |

---

## Option A: Mixamo (Empfohlen)

### Schritte

1. **Mixamo öffnen**
   - URL: https://www.mixamo.com
   - Mit Adobe ID einloggen (kostenlos)

2. **Character auswählen**
   - Tab "Characters" klicken
   - Empfohlene Characters:
     - **Y Bot** (einfach, gut für Prototyping)
     - **X Bot** (weibliche Variante)
     - **Remy** (realistischer)
   - Auf gewünschten Character klicken

3. **Download**
   - "Download" Button klicken
   - Format: **FBX for Unity (.fbx)**
   - Pose: **T-pose**
   - Download starten

4. **Datei speichern**
   - Speichern unter: `Assets/Characters/[CharacterName].fbx`

### Mixamo Character Empfehlungen

| Character | Typ | Polygone | Empfehlung |
|-----------|-----|----------|------------|
| Y Bot | Robot | ~15k | Prototyping |
| X Bot | Robot | ~15k | Prototyping |
| Remy | Human | ~25k | Realistisch |
| Michelle | Human | ~20k | Realistisch |

---

## Option B: Unity Asset Store

### Kostenlose Assets

1. **Unity Asset Store öffnen**
   - Window → Asset Store (oder Browser)

2. **Suche**
   - "humanoid character free"
   - Filter: Preis → Free

3. **Empfohlene Assets**
   - "Starter Assets - Third Person Character Controller" (Unity)
   - Enthält Character + Animationen

4. **Import**
   - Add to My Assets → Open in Unity
   - Import nur den Character (ohne Controller-Code)

---

## Option C: Eigenes Modell

Falls ein eigenes Modell verwendet wird:

### Anforderungen
- Humanoid Skeleton mit korrekten Bone-Namen
- T-Pose als Bind Pose
- Korrekte Skalierung (1 Unit = 1 Meter)

### Export-Einstellungen (Blender)
```
Format: FBX
Scale: 1.0
Apply Modifiers: Yes
Forward: -Z Forward
Up: Y Up
Bake Animation: No (separat)
```

---

## Verifikation

Nach dem Download prüfen:

- [ ] FBX Datei vorhanden
- [ ] Datei in `Assets/Characters/` gespeichert
- [ ] Dateigröße plausibel (1-10 MB)

---

## Dateien nach diesem Schritt

```
Assets/
└── Characters/
    └── YBot.fbx (oder gewählter Character)
```
